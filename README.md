# Ram Bhogesara — Portfolio

Personal portfolio website for Ram Bhogesara, a research-driven Data Science student at IIT Madras focused on applied AI systems, structured problem formulation, and real-world decision systems.

## Live Site

[**bhogesararam23.github.io/my-portfolio-website**](https://bhogesararam23.github.io/my-portfolio-website/)

## Tech Stack

- **React** — Component architecture
- **Vite** — Build tool
- **Tailwind CSS v4** — Utility styles
- **Framer Motion** — Minimal scroll transitions
- **GitHub Pages** — Deployment via GitHub Actions

## Local Development

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173/`

## Build

```bash
npm run build
npm run preview
```

## Deployment

Deployed automatically via GitHub Actions on every push to `main`. The workflow builds the project and deploys the `dist/` folder to GitHub Pages.
